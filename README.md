# Wajha Camp Web Ecosystem

A complete web ecosystem developed for the Wajha Camp at King Saud University to support camp operations, participant management, and event organization.

## Overview

As the Technical Development Team Lead, I led the development of **7+ interconnected web platforms** that supported different aspects of the camp under a unified ecosystem.

## Featured Platforms

- 📝 Registration Platform
- 📊 Admin Dashboard with participant management and CSV export
- 📅 Agenda & Schedule Website
- ⭐ Closing Ceremony Evaluation Platform

...and additional internal platforms developed to support camp operations and organization.

## My Responsibilities

- Delivered the system within organizational deadlines.
